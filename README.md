# 🤖 تریاکی — Teriaky

بات بازی شبیه‌ساز اقتصادی استراتژیک تلگرام با تم دنیای زیرزمینی خیالی — **فاز ۴ (تیم + کوئست + استخراج)**

**پایتون 3.11+ | python-telegram-bot v22 (async) | SQLAlchemy 2 + SQLite**

---

## راه‌اندازی

**لوکال:**

```bash
pip install -r requirements.txt
export TERIAKY_TOKEN="توکن-از-BotFather"
python bot.py
```

**روی Railway:** آموزش کامل قدم‌به‌قدم (ساخت Volume و متغیرها + بک‌آپ) → [RAILWAY_DEPLOY.md](RAILWAY_DEPLOY.md)

تست آفلاین منطق بازی (بدون تلگرام):

```bash
python tests_smoke.py    # 176 تست
```

## ⚠️ برای کار کردن توی گروه

ربات باید پیام‌های متنی گروه رو ببینه تا «حمله» و «کنده کاری تیمی» کار کنه:

1. تو [@BotFather](https://t.me/BotFather) بزن `/setprivacy`
2. رباتت رو انتخاب کن → **Disable**
3. اگه ربات از قبل تو گروهه: **حذفش کن و دوباره ادش کن**

تو گروه دکمه «🏠 منوی اصلی» از پیام‌های ربات حذف میشه که فضای گروه شلوغ نشه

## 🏴 سیستم تیم

- **ساخت تیم** از لول 10 (هزینه 5,000 تی‌پوینت) — بعد «ساخت تیم» ازت اسم می‌خواد و باهاش تیم ساخته میشه
- **جوین** از لول 5 — هر عضو فقط تو یه تیم می‌تونه باشه (سقف 20 نفر)
- **بیوی تیم** (پروفایل تیم) رو رهبر با «ست بیو تیم» ست می‌کنه و تو آمار نمایش داده میشه
- اسم تیم تو پروفایل بازیکن هم میاد 🏴

### 📜 کوئست‌های روزانه گروهی

پیشرفتشون **جمع کل اعضاست** و جایزه **به تک‌تک اعضا** می‌رسه — هر روز UTC ریست میشن:

| کوئست | هدف | جایزه برای هر عضو |
|---|---|---|
| ⚔️ کشتن 25 نفر | 25 برد تو حمله (هرکی بزنه حسابه) | 300 تی‌پوینت |
| 🌾 برداشت 10 محصول | 10 برداشت (هر عضو حسابه) | 200 تی‌پوینت |

استعلام با «کوئست» — کامل شدنش خودش داخل پیام حمله/برداشت اعلام میشه

### ⛏ کنده‌کاری تیمی (استخراج)

«کنده کاری تیمی» رو بزنی استارت/جوین میشی — **70٪ اعضا** باید بزنن تا کامل بشه:

```
⛏ کنده‌کاری تیمی — تیم «فوتبالیست‌ها»
لازمه 70٪ اعضا (5 نفر از 7) دستور رو بزنن
4 نفر از 7 نفر به کنده‌کاری پیوستند
1 نفر تا تکمیل کنده‌کاری
```

کامل که شد یه پول خوب (برای هر پیوسته 150 تا 400) **میره تو خزانه تیم** — هر ساعت یه بار، 5 دقیقه فرصت جمع شدن

موقع لول‌آپ یه پیام تبریک با جایزه‌ها میاد: اسکناس (250 × لول جدید) + شارژ کامل انرژی + لیست چیزایی که آنلاک شدن (بذر/سلاح/زره/سگ/زمین/تیم)

## 💾 بک‌آپ ادمین

- `/backup` → کل اطلاعات بازی به‌صورت یه فایل `.db` توی یه قاب تمیز ارسال میشه
- `/upload_backup` → فایل رو برمی‌گردونی، اعتبارسنجی میشه و اگه سالم بود کل دی‌بی **روی Volume** با فایل جایگزین میشه (ری‌دیپلوی هم نمی‌پره)
- فقط برای آیدی‌های توی `TERIAKY_ADMIN_IDS` — به غریبه هیچ واکنشی نداره

## سیستم حمله

```
شانس برد مهاجم = (حمله پایه + سلاح + سگ‌ها) ÷ (این + دفاع پایه حریف + زره‌اش)
```

- کولدون 1 دقیقه + هزینه انرژی
- برد: سرقت 10٪ تا 25٪ سکه‌های حریف
- مادیفایرها: بونس گرگ سیاه (تا +15٪) → بعد نصف زره افسانه‌ای حریف (50٪−)
- هر برد روی کوئست «کشتن 25 نفر» تیمت حساب میشه 🏴

## سیستم لول (Idle/RPG)

`xp لازم = 50 × لول^1.6` — لول‌های اول سریع و لول‌های بالا سنگین‌تر ولی قابل پیشرفت

اثر لول روی: خرید زمین (هر زمین لول خودشو می‌خواد) | درآمد برداشت (+2٪ هر لول) | قدرت سلاح و زره (+2٪ هر لول) | آیتم‌ها | ساخت تیم (10) | جوین تیم (5)

## توسعه

اضافه کردن آیتم/سگ/بذر جدید = فقط یه خط توی `config.py` — زمین‌ها و تیما و کوئست‌ها هم کاتالوگی‌ان
دیتابیس قدیمی با مایگریشن سبک `database.py` سازگار میشه (ستون‌های جدید خودکار ADD میشن و جداول تیم خودکار ساخته میشن)
