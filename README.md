### 🛡 زره‌ها
کاهش خسارت + یه زره افسانه‌ای 👑:

### 🐕 سگ‌ها
نام + نژاد + لول + XP + قدرت حمله + قابلیت ویژه — قدرتشون به حمله تو اضافه میشه

|---|---|---|---|
| پیتبول رکس | پیتبول | ۸ (+۲ هر لول) | وفادار |

## سیستم حمله

```
شانس برد مهاجم = (حمله پایه + سلاح + سگ‌ها) ÷ (این + دفاع پایه حریف + زره‌اش)
```

- کولدون ۱ دقیقه + هزینه انرژی
- برد: سرقت ۱۰٪ تا ۲۵٪ سکه‌های حریف
- مادیفایرها: بونس گرگ سیاه (تا +۱۵٪) → بعد نصف زره افسانه‌ای حریف (۵۰٪−)
- تو گروه: ریپلای + «حمله» | تو PV: هدف رندوم از منوی ⚔️

## سیستم لول (Idle/RPG)

`xp لازم = ۵۰ × لول^۱.۶` — لول‌های اول سریع (۵۰ | ۱۵۱ | ۲۸۹) و لول‌های بالا سنگین‌تر ولی قابل پیشرفت

اثر لول روی: خرید زمین (زمین n لول n می‌خواد) | درآمد برداشت (+۲٪ هر لول) | محصولات قابل کاشت | قدرت سلاح و زره (+۲٪ هر لول) | آیتم‌های فروشگاه

## توسعه

اضافه کردن آیتم/سگ/بذر جدید = فقط یه خط توی `config.py` — لول موردنیاز و قیمت و توضیح و قابلیتشو بده، بقیه‌ش اتوماتیکه
دیتابیس قدیمی هم با مایگریشن سبک `database.py` سازگار میشه (ستون‌های جدید خودکار ADD میشن)

## نکته دکمه‌های رنگی
