# 🤖 تریاکی — Teriaky

هسته اولیه ربات بازی شبیه‌ساز اقتصادی استراتژیک تلگرام با تم دنیای زیرزمینی خیالی

**پایتون ۳.۱۱+ | python-telegram-bot v22 (async) | SQLAlchemy 2 + SQLite**

---

## راه‌اندازی

**لوکال:**

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
export TERIAKY_TOKEN="توکن-از-BotFather"
python bot.py
```

**روی Railway:** آموزش کامل قدم‌به‌قدم (ساخت Volume و متغیرها) → [RAILWAY_DEPLOY.md](RAILWAY_DEPLOY.md)

برای PostgreSQL فقط `TERIAKY_DB` رو عوض کن (نمونه توی `.env.example`)

تست آفلاین منطق بازی (بدون تلگرام):

```bash
python tests_smoke.py
```

## دکمه‌های رنگی تلگرام

از فیچر **Button styles** تلگرام (Bot API 9.4) استفاده شده — نیازمند `python-telegram-bot>=22.7`

| استایل API | رنگ | کاربرد تو بازی |
|---|---|---|
| `primary` | آبی | اکشن‌های اصلی (منو | خرید | آپگرید | پیدا کردن هدف) |
| `success` | سبز | ✅ تایید و برداشت |
| `danger` | قرمز | ❌ لغو |

هر اکشن مهم اول یه پیام تایید می‌گیره با کیبورد `✅ تایید / ❌ لغو` و بعد با callbackای با پیشوند `cf:` اجرا میشه

## ساختار callback_data

```
menu:home | menu:profile | menu:farm | menu:shop | menu:attack | menu:rank
farm:buy                    → cf:farm:buy
farm:plant:<plot_id>        → انتخاب محصول
farm:plant:<plot_id>:<crop> → cf:plant:<plot_id>:<crop>
farm:hv:<plot_id>           → برداشت (مستقیم چون هزینه نداره)
farm:up:<plot_id>           → cf:farm:up:<plot_id>
shop:buy:<item_key>         → cf:shop:buy:<item_key>
att:find                    → کارت هدف + cf:att:<target_id>
cl                          → لغو همه‌چیز
noop:<context>              → دکمه‌های اطلاعاتی
```

## گیم‌پلی فاز اول

### واحد پول

**تی‌پوینت** — تو متن پیام‌ها کامل میاد و تو دکمه‌ها و لیست‌های خلاصه **TP** نوشته میشه
برای تغییرش فقط `UNIT` و `UNIT_SHORT` رو توی `utils.py` عوض کن

### محصولات (به ترتیب باز شدن با لول)

| محصول | لول | کاشت | زمان | فروش پایه |
|---|---|---|---|---|
| تریاک | ۱ | ۱۵۰ TP | ۵ دقیقه | ۴۲۰ TP |
| ماری‌جوانا | ۳ | ۴۰۰ TP | ۸ دقیقه | ۱٬۱۵۰ TP |
| کوکا | ۵ | ۱٬۰۰۰ TP | ۱۲ دقیقه | ۳٬۰۰۰ TP |
| قات | ۷ | ۲٬۴۰۰ TP | ۱۸ دقیقه | ۷٬۵۰۰ TP |
| پیوته | ۱۰ | ۶٬۰۰۰ TP | ۲۵ دقیقه | ۲۰٬۰۰۰ TP |

زمین لول ۲ و ۳ درآمد رو ×۱.۸ و ×۲.۸ می‌کنه و سرعت رشد رو بهتر — همه اعداد توی `config.py`

### کنده کاری

کلمه «کنده کاری» رو هر ۲۴ ساعت بفرست — جایزه بین **۱۰ تا ۱۵۰ تی‌پوینت** با قرعه وزن‌دار (**۷۵٪ مواقع بین ۱۰ تا ۱۰۰**)
به صورت دستور `/mine` هم هست (دستور فارسی توی تلگرام ساپورت نمیشه پس متن آزاد هم گذاشته شده)
درآمد مزرعه عمدا خیلی بالاتر از کنده‌کاریه که تناسب اقتصاد حفظ بشه

### حمله

- هدف رندوم تو بازه ±۲ لول خودت
- شانس برد مهاجم = حمله ÷ (حمله + دفاع) — بعد از ضریب شانس ۰.۸۵ تا ۱.۱۵ رو قدرت‌ها
- برد: سرقت ۱۰٪ تا ۲۵٪ پول نقد قربانی + تجربه
- باخت: سوختن انرژی اضافه
- کولدون ۱۰ دقیقه + هزینه انرژی هر حمله

### فروشگاه

۵ سلاح و ۴ زره با قیمت و بونس متفاوت — فقط بهترین سلاح و بهترین زره روی استت‌ها حساب میشه
